# 八字生成器

一个基于传统八字命理学的在线工具，可以根据出生信息推算四柱八字，并提供简要的运势分析。

## 功能特点

- **四柱八字排盘**：根据出生年月日时自动计算年柱、月柱、日柱、时柱
- **五行分析**：显示每个柱的天干地支及其对应的五行属性
- **流年运势**：基于当前年份的流年干支，分析本年的事业运、财运、桃花/婚姻运
- **简洁界面**：现代化的 UI 设计，响应式布局，支持移动端访问
- **数据验证**：完整的表单验证，确保输入数据的合法性

## 使用方法

1. 打开 `index.html` 文件（可直接在浏览器中打开，无需服务器）
2. 填写基本信息：
   - 姓名（可选）
   - 性别（男/女）
   - 出生年份（1900-2100）
   - 出生月份（1-12）
   - 出生日（1-31）
   - 出生时（0-23）
   - 出生分钟（0-59，可选）
   - 出生地（可选）
3. 点击"生成八字"按钮
4. 查看排盘结果和运势分析

### 核心算法

- **年柱计算**：以 1984 年（甲子年）为基准，通过年份偏移计算天干地支
- **月柱计算**：根据年干和月份，使用"五虎遁"法推算月干
- **日柱计算**：以 1900-01-31（甲子日）为基准，通过日期差计算
- **时柱计算**：根据日干和时辰，使用"五鼠遁"法推算时干
- **运势分析**：基于五行生克关系，分析流年对命局的影响
